🏥 Hospital Management System

A console-based Hospital Management System developed in C to simplify hospital operations by efficiently managing patients, doctors, appointments, and billing. This project was developed collaboratively as part of an academic curriculum.

✨ Features

- Patient Management
- Doctor Management
- Appointment Scheduling
- Billing System
- Search, Update & Delete Records

🛠️ Technologies

- C Programming
- Structures
- Functions
- Arrays
- File Handling

🚀 How to Run

1. Clone the repository.
2. Compile "main.c".
3. Run the executable.
4. Follow the on-screen menu.
